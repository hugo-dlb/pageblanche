# Pageblanche
A single html page that anyone can edit at the same time.

# How it works
Users can send one "action" every 30 seconds.
The server will execute the most voted action, e.g. write the character "a" at line 3 and column 10.

# Technologies
- Node.js (back-end)
- Angular (front-end)
